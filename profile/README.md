## Introduction

未来绿动是一款专为中等碳排放企业设计的的碳管理系统，针对低端能浪费严重、碳核算能力不足问题，提供节能物联网设备，并搭配碳核算应用APP和碳监测大屏，助力中等碳排放企业实现节能减排目标。
